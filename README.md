# Hospital-Emergency-Room-Dashboard-Excel-
Interactive Hospital Emergency Room Dashboard built in Microsoft Excel. This project analyzes patient admission status, age distribution, wait time, gender analysis, and department referrals using Pivot Tables, Pivot Charts, Slicers, KPIs, and interactive visualizations to generate meaningful healthcare insights.

#📊 Key Features
1.Patient Admission Status (Admitted vs. Not Admitted)
2.Patient Age Distribution
3.Patient Gender Analysis
4.Patient Wait Time Analysis (Within 30 Minutes)
5.Department Referral Analysis
6.Monthly and Yearly Interactive Slicers
7.Key Performance Indicators (KPIs)
8.Interactive Charts and Visualizations

🛠️ Tools Used
Microsoft Excel
Pivot Tables
Pivot Charts
Slicers
Conditional Formatting
Data Cleaning
Data Analysis
Dashboard Design

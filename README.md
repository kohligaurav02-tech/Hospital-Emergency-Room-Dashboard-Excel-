# Hospital-Emergency-Room-Dashboard-Excel-
Interactive Hospital Emergency Room Dashboard built in Microsoft Excel. This project analyzes patient admission status, age distribution, wait time, gender analysis, and department referrals using Pivot Tables, Pivot Charts, Slicers, KPIs, and interactive visualizations to generate meaningful healthcare insights.

📊 Key Features
Patient Admission Status (Admitted vs. Not Admitted)
Patient Age Distribution
Patient Gender Analysis
Patient Wait Time Analysis (Within 30 Minutes)
Department Referral Analysis
Monthly and Yearly Interactive Slicers
Key Performance Indicators (KPIs)
Interactive Charts and Visualizations

🛠️ Tools Used
Microsoft Excel
Pivot Tables
Pivot Charts
Slicers
Conditional Formatting
Data Cleaning
Data Analysis
Dashboard Design
